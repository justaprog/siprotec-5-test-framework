# Frontend Architecture
## Folder Structure
The frontend code is organized into the following main folders:
- `src`: Contains the main source code for the frontend application.
  - `components`: Reusable Vue components used across the application.
  - `views`: Contains full page components that are mapped to routes.
  - `services`: Modules responsible for making API calls to the backend services.
  - `router`: Vue Router configuration for handling navigation between views.
  - `types`: Contains TypeScript interfaces for frontend data models.
